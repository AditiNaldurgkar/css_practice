Lets learn about some new words..

1] Metaverse : A virtual reality environment in which individuals can engage with a computer-generated world as well as other users.
2] Coulrophobia : extreme or irrational fear of clowns
3] Mislay : unintentionally put an object where it cannot readily be found and so lose it temporarily
